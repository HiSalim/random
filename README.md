# Random
Obtiene un número pseudoaleatorio entre los valores ==[0, 1]==
y muestra por ==stdout== el mensaje de si es mayor o menor de ==0.5==.

## Example
==Modo de ejecución==
`HiSalim@amazigh:$> nodejs random.js`

==Salida==
`HiSalim@amazigh:$>0.9669474302791059 MAYOR que 0,5
`
